# v0.9.0-beta.1 邀请内测候选说明

`v0.9.0-beta.1` 是绘卖 AI 的小范围邀请内测候选，不是公开注册版或 GA 稳定版。

## 本候选包含

- 邀请制登录、商户隔离、运营后台与用量/配额门禁。
- 脚本、图片、视频、TTS 与合成链路的失败关闭、限流和上传空间保护。
- 模型修订证据、Golden Set 评测入口、调用量/成本记录和发布回滚约束。
- 备份/恢复、生产预检、边缘上传限制、Docker 源码边界与供应链证据。
- 完整 commit SHA 版本身份和预发布标签防误发门禁。

## 放行边界

本地 lint、类型、单测、生产构建、standalone 和备份恢复通过，只能证明候选代码具备进入外部验证的条件。首位外部用户在以下证据齐备前仍为 **No-Go**：

- 远程 CI 、已测 Docker 镜像与 GHCR digest/来源证明。
- 真实 HTTPS/Nginx 正反上传上限探针。
- 主备模型的真实 Golden Set、费用和故障切换证据。
- 真实写入冻结窗口、站外备份版本回执和生产预检零 `FAIL`。

放行判定以 `产品文档/17_邀请内测放行报告_2026-07-16.md` 为准，不得用本说明替代真实证据。

## 升级与回滚

1. 发布前按运行手册冻结写入，完成并校验站外备份。
2. 只部署与已通过冒烟的完整 commit SHA 绑定的镜像 digest，不使用浮动标签。
3. 回滚时回到上一个已核验 digest；如涉及数据兼容性，必须同时执行已演练的备份恢复流程。

详细操作见 `产品文档/邀请内测上线运行手册.md`。
